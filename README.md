# 🎓 Pós Tech AI Scientist — FIAP
Este repositório reúne os projetos desenvolvidos ao longo da Pós Tech de AI Scientist da FIAP, abrangendo atividades práticas, tech challenges e aplicações de Machine Learning, Data Science e Inteligência Artificial.
O objetivo é centralizar os trabalhos, documentar a evolução técnica e servir como portfólio acadêmico e profissional.

---
## 📚 Sobre a Pós Tech AI Scientist
A Pós Tech AI Scientist da FIAP é focada no desenvolvimento de competências em:

- Análise e exploração de dados
- Machine Learning supervisionado e não supervisionado
- Modelos preditivos e avaliação
- Boas práticas de engenharia de ML
- Deploy e aplicações práticas de IA


Os projetos deste repositório refletem a aplicação desses conceitos em cenários reais e desafios técnicos.

## 📁 Descrição das pastas
- tech-challenge-*: Projetos avaliativos e desafios técnicos da pós
- notebooks/: Análises exploratórias, experimentos e estudos guiados
- estudos/: Materiais complementares, provas de conceito e exercícios
- datasets/: Bases de dados utilizadas nos projetos (quando aplicável)


Cada projeto possui seu próprio README.md com detalhes específicos.

## 🛠️ Tecnologias Utilizadas
Ao longo dos projetos, são utilizadas principalmente:

- Python 3.9+
- Pandas, NumPy
- Scikit-learn
- LightGBM / XGBoost
- Matplotlib, Seaborn
- Streamlit
- MLflow
- Git & GitHub
